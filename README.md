# GikoLLVM

## これは何

昔懐かしい萌え言語「ギコBASIC」ライクな言語のLLVMによる実装です。

## 必要なもの

- CMake
- LLVM+Clang
- Boost C++ Libraries
- 転んでも泣かない心

## 動作確認環境

- Gentoo Linux amd64
- LLVM+Clang 3.5
- Boost C++ Libraries 1.55

## ビルド方法

```console
$ cd build
$ cmake ..
$ make
```

## テスト方法

```console
$ cd build
$ ./test.sh
```
